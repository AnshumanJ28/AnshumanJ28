## Featured Projects

### Chess Engine with AlphaZero-style AI
A fully functional chess engine built from scratch in Python, powered by a custom **Neural Network + Monte Carlo Tree Search (MCTS)** — inspired by DeepMind's AlphaZero.

**What's under the hood:**
- `chesseng.py` — Complete chess rules engine: move generation, castling, en passant, pawn promotion, check/checkmate/stalemate detection
- `NeuralNet.py` — Custom deep residual network (10 ResBlocks) with a **policy head** (move probabilities) and **value head** (position evaluation)
- `mcts.py` — Full MCTS implementation with UCB scoring, Dirichlet noise for exploration, and value backup
- `BoardEncoder.py` — Encodes board state into an 18-plane tensor (piece positions, castling rights, en passant, turn)
- `Train.py` — Self-play training loop
- `main.py` — Game entry point

[View Repository](https://github.com/AnshumanJ28/AlphaZ0)

### Diabatic-B5 — Retinal Disease Classification
Binary classification of retinal diseases from fundus images using **EfficientNet-B5**.
Trained on APTOS 2019 dataset with generalization testing on RFMiD.
Achieved strong ROC-AUC with a two-phase fine-tuning strategy.
